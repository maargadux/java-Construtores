# 🧠 Java - Lógica de Construtores

Projeto em Java focado no aprendizado de construtores, sobrecarga e modelagem básica de objetos utilizando Programação Orientada a Objetos (POO).

## 🚀 About the Project

This project demonstrates how to create and use constructors in Java, including constructor overloading and object instantiation.

It was built as part of my studies in Java fundamentals, focusing on clean code structure and understanding how objects are initialized.

## 📚 Concepts Covered

* Constructors in Java
* Constructor Overloading
* Object Creation
* Basic Object-Oriented Programming (OOP)
* Attributes and Methods

## 🛠️ Technologies

* Java
* Eclipse

## 📂 Project Structure

```bash
src/
 └── com/java/
     ├── Produto.java
     └── ProdutoTestar.java
```

## 💡 Example

```java
Produto p1 = new Produto("Caneta Preta", "Bic", 1.65f);
System.out.println(p1.nome);
```

The goal of this project is to strengthen my understanding of Java OOP fundamentals and prepare for backend development using Java and Spring Boot.
